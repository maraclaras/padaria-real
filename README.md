# Padaria Real

O projeto tem como cliente uma padaria local e propõe o desenvolvimento de uma plataforma de gestão voltada para o controle de produtos e desperdício alimentar.

O sistema permitirá que o administrador da padaria registre e acompanhe informações essenciais, como:

* Produtos disponíveis no estoque e no balcão de vendas.

* Datas de vencimento de cada item, com alertas para produtos próximos do prazo.

* Taxa de desperdício, calculada a partir dos produtos descartados.

* Perda financeira associada ao desperdício, por meio de uma calculadora que relaciona o valor de custo e venda dos itens.

Com esses dados, a padaria poderá ter maior visibilidade sobre seu estoque, tomar decisões mais estratégicas para reduzir perdas e implementar ações sustentáveis de reaproveitamento ou desconto em produtos próximos ao vencimento.

Dessa forma, o sistema contribui não apenas para a redução do impacto ambiental, diminuindo o descarte de alimentos, mas também para a otimização financeira do negócio, unindo tecnologia e sustentabilidade na gestão cotidiana da padaria.
